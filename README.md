# Study Planner Agent 📚🤖

A Django web app where an AI agent (powered by Gemini) builds you a personalized
study plan, tells you what to focus on today, and quizzes you on topics —
built for the Kaggle "5-Day AI Agents Intensive" Capstone Project.

## What it does
1. **Add subjects** with exam dates and difficulty.
2. **Generate Study Plan** — the agent (Gemini) creates a day-by-day plan,
   prioritizing harder subjects and nearer exam dates.
3. **Today's Focus** — the agent gives you a short, motivating summary of
   exactly what to study today.
4. **Quiz Me** — the agent generates 3 quick quiz questions on any subject/topic.

## How it maps to "AI Agent" concepts (for your capstone writeup)
- **Model** → Gemini 2.5 Flash
- **Tools** → Date calculations, database read/write (Django ORM)
- **Memory** → Study plan stored in SQLite database
- **Task/Orchestration** → `agent.py` decides what prompt to send and how
  to parse Gemini's response into structured study tasks

## Setup Instructions (Windows)

### 1. Extract the zip
Extract `study_planner_agent.zip` anywhere, e.g. `D:\study_planner`

### 2. Open folder in Antigravity IDE or any terminal
```
cd D:\study_planner\study_planner
```

### 3. Install dependencies
```bash
pip install django google-genai
```

### 4. Set your Gemini API key
You already have this from Google AI Studio (`aistudio.google.com/apikey`).

**PowerShell (for current session):**
```powershell
$env:GEMINI_API_KEY="paste_your_key_here"
```

**Or set it permanently (then restart your terminal):**
```powershell
setx GEMINI_API_KEY "paste_your_key_here"
```

### 5. Run migrations (creates the database)
```bash
python manage.py migrate
```

### 6. Start the server
```bash
python manage.py runserver
```

### 7. Open in browser
Go to: http://127.0.0.1:8000/

## Using the app
1. Click **"+ Add Subject"** — add each exam subject with its date and difficulty.
2. Click **"🤖 Generate Study Plan"** — the agent creates your full schedule.
3. Click **"Today's Focus"** in the nav bar — see what to study today with an
   encouraging message from the agent.
4. Click **"Quiz Me"** — pick a subject (and optionally a topic) to get a
   quick 3-question quiz generated by the agent.
